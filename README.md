# 🧱 Python Project Template

A clean, reusable, and scalable Python project structure. Ideal for general-purpose Python applications, data projects, and development best practices.

---

## 🚀 Quick Start
Requiremetns:
Python>=3.12
UV package manager to be installed.

### 🔧 Setup in 5 Steps

1. **Clone the Repository**
    ```bash
    git clone https://github.com/NitinShelke/project_template.git
    cd project_template
    ```
2. change the name of project as per your choice and modify the same in pyproject.toml

3. **Create Virtual Environment**
   use commands :
   1. uv sync
   

4. **Install Dependencies**
     1. uv pip install -e .["dev"]
     2. pre-commit install
